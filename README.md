# ORBIT FORGE

AI-powered DevOps automation and continuous delivery platform powered by MiMo V2.5 + Hermes Agent.

## Architecture
- **Planner** — Sprint planning & task decomposition
- **Builder** — Automated build & dependency management
- **Tester** — AI-generated test suites & coverage analysis
- **Deployer** — Zero-downtime deployment orchestration
- **Monitor** — Production health monitoring & rollback triggers

## Tech Stack
- MiMo V2.5 (chain-of-thought deployment reasoning)
- Hermes Agent (orchestration layer)
- Claude series (code analysis)
